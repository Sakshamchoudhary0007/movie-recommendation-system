# 🎬 Movie Recommendation System

## 📌 Overview

This project is a Content-Based Movie Recommendation System built using Machine Learning techniques. It recommends movies similar to the one selected by the user.

## 🚀 Features

* Recommend similar movies instantly
* Simple and interactive UI using Streamlit
* Uses cosine similarity for recommendations

## 🧠 How It Works

* Movie data is processed and converted into vectors
* Similarity between movies is calculated using cosine similarity
* Top similar movies are recommended

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ⚠️ Note

Model files (.pkl) are not uploaded due to GitHub size limitations. They can be regenerated using the training script.

## 📷 Output

The system provides movie recommendations based on selected input.
